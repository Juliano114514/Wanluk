# 语言学田调录音助手 Android App（暂名）

## 1 基本设计

- 参考古早的**北语录音**软件，实现基本的字例导入 录音 文件管理等基本功能
- 做一个傻瓜式的显示、录音、与跳转下一词，以及录音的打包和分享操作
- 类似于**不背单词**那样简约的设计：字例展示、例词、录音控制、下一字/词 几个基本的按钮
- 本地预设字例按切韵韵部分类，如端透泥知彻澄等
- 分享/导入 短期优先实现本地文件级的导入 / 导出（如本地 JSON/CSV 字例库、录音文件包）

## 2 功能设计

### 2.2 录音业务模块

#### 2.2.1 录音工具选型

- 核心方案：Android 系统 MediaRecorder API（基础录音）+ AudioRecord API（高精度录音）

  - 适配策略：根据设备系统版本自动切换（Android 10 + 优先使用 MediaRecorder 的新特性，低版本兼容传统 API）；
  - 第三方插件：仅作为降级备选（如 SoundRecorder），优先保证系统 API 的稳定性与适配性；
  - 录音参数：默认采样率 44.1kHz、位深 16bit、单声道（兼顾文件大小与音质，满足语言学分析需求），支持自定义参数（二期）。

  

#### 2.2.2 录音文件管理

- 命名规则（唯一标识）：

  ```
  [年月日时分秒]_[韵摄]_[例字]_[录音次数].m4a
  ```

  - 示例：`20240520153020_通_东_1.m4a`（2024 年 5 月 20 日 15:30:20 录制、通摄、例字东、第一次录制）；

  

- 存储结构（本地目录）：

  ```
  /Android/data/com.linguistics.recorder/files/
  ├── record/          # 所有录音文件
  │   ├── 20240520/    # 按日期分目录
  │   │   ├── 通摄/    # 按韵摄分目录
  │   │   │   ├── 20240520153020_通_东_1.m4a
  │   │   │   └── ...
  │   │   └── ...
  │   └── ...
  └── wordlist/        # 字例库文件
      ├── default/     # 内置默认字例库
      └── custom/      # 自定义导入字例库
  ```

- 分类维度：支持按「韵部 / 摄」「录制日期」「录制人」（自定义标签）筛选录音文件；

- 重录逻辑：同一字例多次录音保留所有版本，标记「最新版」，支持一键删除冗余录音。

#### 2.2.3 录音结果分享与上传

- 打包分享（一期核心）：

  - 支持按「韵摄」「日期」「自定义选择」打包录音文件为 ZIP；
  - 分享渠道：系统原生分享（微信 / QQ / 邮件 / 本地存储），无中台依赖；

- 后端交互（二期规划）：

  - 暂不接入中台，仅预留上传接口；
  - 风险预案：若后续接入上传，需实现「文件 MD5 校验」（避免重复上传）、「分片上传」（适配大文件）、「失败重传」（网络容错）；DDoS 风险依赖后端防护，客户端仅做请求频率限制。

  

### 2.3 录音界面设计

#### 2.3.1 基础界面布局（核心）

|    区域    | 占比 |                         内容 / 功能                          |
| :--------: | :--: | :----------------------------------------------------------: |
| 例字展示区 | 40%  |             大号字体展示核心例字，居中高亮显示；             |
| 组词展示区 | 15%  |      例字下方小号字体展示组词（多个组词用「/」分隔）；       |
| 录音控制区 | 25%  | 「录音 / 暂停」「重录」按钮（大尺寸，易点击），显示当前录音时长； |
|   导航区   | 20%  | 「上一字 / 词」「下一字 / 词」按钮，显示当前进度（如「3/50」）； |

- 交互逻辑：录音完成后自动聚焦「下一字 / 词」按钮，点击后清空录音状态，加载下一字例，支持一键连续录音。

#### 2.3.2 拓展功能

- 拓展 1：多字并列展示模式

  - 支持自定义并列数量（2-4 字），如「东 冻 通 桶」横向排列；
  - 每个字例独立显示组词，录音按钮支持「单字录音」「批量录音」（一次性录制所有并列字例）；

  

- 拓展 2：读词模式

  - 自动选取每个字例「组词」字段的第一个词，优先展示组词（例字作为副标题）；
  - 录音逻辑适配：按词录音，文件命名同步替换为组词（如`20240520153020_通_东风_1.m4a`）；

  

- 拓展 3：快捷操作

  - 录音时支持音量键快捷控制（音量 +：暂停，音量 -：重录）；
  - 双击例字展示区，快速查看该字例的完整韵部信息（聲 / 呼 / 等 / 韻 / 調等）；

  

- 拓展 4：状态反馈

  - 录音中显示动态波形图，直观反馈录音状态；
  - 录音完成后自动校验文件完整性，异常时提示重录。

  

### 2.4 兼容性与适配（补充）

- 系统版本适配：最低支持 Android 8.0（API 26），覆盖 95% 以上安卓设备；

- 机型适配：针对不同屏幕尺寸（手机 / 平板）做界面自适应，保证按钮可点击、文字清晰；

- 权限处理：

  - 核心权限：录音权限（必选）、存储权限（可选，Android 11 + 采用媒体库 API 规避）；
  - 权限引导：首次打开 APP 时分步引导授权，拒绝后提供手动授权指引。

  

### 2.5 二期规划（补充）
